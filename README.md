# FleetFlow – Fleet Management System

## Overview
FleetFlow is a full-stack fleet & logistics management system built during the Odoo Hackathon. 
It helps manage vehicles, drivers, trips, and maintenance records through a centralized dashboard.

## Key Features
- Vehicle & Driver Management
- Trip Lifecycle Tracking
- Maintenance Logs
- Admin Dashboard
- Secure Authentication

## Tech Stack
Frontend: HTML, CSS, Bootstrap, JavaScript  
Backend: Flask  
Database: MySQL
